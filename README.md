# کانون تبلیغاتی حنان - نسخه HTML

این پروژه نسخه HTML از وب‌سایت کانون تبلیغاتی حنان است که از Next.js به HTML خالص تبدیل شده است.

## ویژگی‌ها

- طراحی ریسپانسیو و مدرن
- پشتیبانی کامل از زبان فارسی و راست‌چین (RTL)
- سیستم احراز هویت با Supabase
- داشبورد ادمین
- رابط کاربری زیبا و کاربرپسند

## فایل‌های پروژه

- `index.html` - صفحه اصلی
- `login.html` - صفحه ورود
- `register.html` - صفحه ثبت‌نام
- `admin-dashboard.html` - داشبورد ادمین
- `styles.css` - فایل استایل‌ها
- `public/next.svg` - لوگو

## نحوه استفاده

1. تمام فایل‌ها را در یک پوشه قرار دهید
2. فایل `index.html` را در مرورگر باز کنید
3. برای استفاده از قابلیت‌های احراز هویت، اتصال اینترنت لازم است

## قابلیت‌ها

### صفحه اصلی
- نمایش اطلاعات کانون تبلیغاتی حنان
- دکمه‌های ورود و ثبت‌نام
- طراحی مدرن و ریسپانسیو

### سیستم احراز هویت
- ثبت‌نام کاربران جدید
- ورود کاربران موجود
- اعتبارسنجی فرم‌ها
- اتصال به Supabase

### داشبورد ادمین
- نمایش آمار کاربران
- لیست تمام کاربران
- مدیریت کاربران

## تنظیمات Supabase

پروژه از Supabase برای احراز هویت استفاده می‌کند. برای استفاده کامل:

1. یک پروژه Supabase ایجاد کنید
2. کلیدهای API را در فایل‌های HTML جایگزین کنید
3. قوانین RLS مناسب تنظیم کنید
